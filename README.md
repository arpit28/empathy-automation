This project contains automated tests for the Empathy client registration flow using Playwright and TypeScript.

## Installation
###  1. Install Node.js

Make sure you have Node.js 18 or higher installed.

Check your version:

```node -v```

### 2. Install project dependencies

From the project root folder, run:

```npm install```

### 3. Install Playwright browsers
```npx playwright install```

## How to Run the Tests
Run all tests (headless)
```npm test```

## Run tests in headed (visible) browser
```npm run test:headed```

## View HTML report

After running tests, open:

```npm run test:report```


## Requirements

Node.js 18+

npm (comes with Node)
